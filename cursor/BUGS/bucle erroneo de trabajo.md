

## Qué hacer cuando la IA entra en bucle

1. Detener inmediatamente: reconoce el bucle y para.
2. Reformular: si la misma indicación falla 3 veces, cambia el enfoque.
3. Simplificar: reduce requisitos y concéntrate en lo esencial.
4. Decir "no puedo hacerlo así": si no funciona, propón una alternativa.
5. Documentar: anota qué no funcionó para evitarlo después.

## Cómo se resolvió problema de scripts con fromato incorrecto que añadía indicaiones dentro?

El problema: el formato pedido (script + frase fuera + "OK?") hacía que al copiar el bloque también se copiara el texto externo, causando errores.

Solución: dejar solo el script dentro del bloque. La frase explicativa y el "OK?" se omiten para evitar que se copien junto al código.
