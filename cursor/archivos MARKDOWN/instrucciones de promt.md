## Instrucciones de uso

1. Copia el prompt completo (versión larga o corta).
2. Reemplaza las partes entre `[CORCHETES]` con la información de tu proyecto.
3. Pega el prompt en Agent mode de Cursor.
4. Ejecuta paso a paso, verificando cada uno antes de continuar.
