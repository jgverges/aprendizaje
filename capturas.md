SOLUCIONES PARA USAR CHAT (NO AGENT):

1. Menú: View > Chat
2. Command Palette (Ctrl+Shift+P) → buscar "Chat" o "Open Chat"
3. Clic en icono de chat en barra lateral izquierda (si existe)

CONFIRMACIÓN DE QUE ESTÁS EN CHAT:
- NO debe decir "Agent" en ningún lugar
- Panel lateral IZQUIERDO simple
- NO tiene dropdowns "∞ Agent" ni "Auto"
- NO tiene botones @ ni opciones avanzadas

CONFIRMACIÓN DE QUE ESTÁS EN AGENT (CONSUME MÁS):
- SÍ dice "∞ Agent"
- Panel lateral DERECHO "Agents"
- Tiene "Plan, @ for context, / for commands"

ATAJOS ACTUALES (ambos abren Agent):
- Ctrl+L → Agent
- Ctrl+Shift+L → Agent
