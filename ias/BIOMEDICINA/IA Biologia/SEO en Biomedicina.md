## SEO académico en biomedicina

### ¿Qué es SEO académico?

Optimización para que tu paper sea encontrado en:
- PubMed/Medline
- Google Scholar
- Bases de datos académicas
- Motores de búsqueda científicos

Objetivo: aumentar visibilidad, citas e impacto.

### 1. SEO académico (optimización para búsqueda)

#### Elementos clave:

| Elemento | Dónde incluir | Ejemplo (VIH/Senescencia) |
|----------|---------------|---------------------------|
| **Palabras clave MeSH** | Abstract, Keywords | "HIV", "Cellular Senescence", "Antiretroviral Therapy" |
| **Términos específicos** | Título, Abstract | "Senescent cells", "HIV reservoir", "Telomere length" |
| **Sinónimos relevantes** | Abstract, Introducción | "Aging" + "Senescence", "AIDS" + "HIV" |
| **Título descriptivo** | Título | Incluir: enfermedad, método, resultado clave |

#### Checklist SEO:

- [ ] Título incluye términos de búsqueda clave
- [ ] Abstract con 3-5 términos MeSH relevantes
- [ ] Keywords: 5-8 términos (MeSH + específicos)
- [ ] Abstract menciona términos clave 2-3 veces
- [ ] Introducción usa terminología estándar del campo

### 2. Factores de publicación (orden de importancia)

#### A. Calidad científica (crítico)

| Aspecto | Qué verifica el revisor |
|---------|------------------------|
| **Originalidad** | ¿Aporta conocimiento nuevo? |
| **Rigor metodológico** | ¿Métodos sólidos y reproducibles? |
| **Significancia** | ¿Relevancia clínica/científica? |
| **Estadística** | ¿Análisis correctos y apropiados? |

---

#### B. Selección de revista (crítico)

| Criterio | Qué buscar |
|----------|------------|
| **Scope** | ¿Publica tu área (VIH/senescencia)? |
| **Impact Factor** | ¿Adecuado para tu trabajo? |
| **Tiempo revisión** | ¿Rápido o puede esperar? |
| **Acceso abierto** | ¿Aumenta visibilidad? |

---

#### C. Redacción y estructura (crítico)

| Sección | Requisitos |
|---------|------------|
| **Abstract** | Claro, completo, SEO optimizado |
| **Introducción** | Contexto, gap, objetivo |
| **Metodología** | Reproducible, detallada |
| **Resultados** | Objetivos, bien presentados |
| **Discusión** | Interpretación, limitaciones, relevancia |
| **Referencias** | Actualizadas, relevantes, formato correcto |
