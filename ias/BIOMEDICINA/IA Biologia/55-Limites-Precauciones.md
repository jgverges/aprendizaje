## 3. Qué NO hacer: límites de las herramientas IA

### ⚠️ No usar ChatGPT para:

- Generar citas bibliográficas (inventa citas falsas)
- Extraer datos de papers (puede inventar datos)
- Redactar resultados (puede inventar resultados)
- Verificar información médica (puede estar desactualizada)

### ✅ Usar herramientas especializadas:

- **Datos verificables** → Elicit (extrae datos reales)
- **Búsqueda de literatura** → Elicit, Research Rabbit (acceso real a papers)

**Regla general**: No usar ChatGPT para datos verificables, citas o resultados. Usar herramientas especializadas.
