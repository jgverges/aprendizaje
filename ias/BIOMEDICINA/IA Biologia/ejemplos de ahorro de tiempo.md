### Cuándo NO usar ChatGPT

| Caso | Por qué no |
|------|------------|
| **Generar citas bibliográficas** | Inventa citas falsas (PMID, DOI, autores) |
| **Extraer datos de papers** | Puede inventar datos que no están en el paper |
| **Redactar resultados** | Puede inventar resultados no reales |
| **Búsqueda en PubMed** | No accede a bases de datos en tiempo real |
| **Verificar información médica** | Puede dar información desactualizada o incorrecta |
| **Análisis de imágenes médicas** | No es herramienta de diagnóstico |
| **Diseñar experimentos críticos** | Requiere validación humana experta |
| **Revisión ética** | Requiere juicio humano especializado |

**Regla general:** No usar ChatGPT para datos verificables, citas, resultados o información que requiera fuentes primarias. Usar Elicit, Zotero o herramientas especializadas.

---

**Resumen:** ChatGPT es útil para redacción, estructura y brainstorming. No para datos, citas o información verificable.
