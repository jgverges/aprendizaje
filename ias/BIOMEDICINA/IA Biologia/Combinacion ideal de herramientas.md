##  Combinaciones ideales de herramientas
**Stack mínimo:**
1. Zotero (referencias)
2. Prism (estadística)
3. Cursor (redacción)
4. Elicit (extracción)
5. BioRender (ilustraciones)

### Escenario 1: Conjunto reducido (mínimo cambio de tecnología)

| Herramienta | Reemplaza | Uso Principal |
|-------------|-----------|---------------|
| **Zotero** | Mendeley | Gestión referencias |
| **GraphPad Prism** | - | Análisis estadístico |
| **Cursor** | - | Redacción, tablas, organización |
| **Elicit** | - | Extracción datos papers |
| **BioRender** | - | Ilustraciones biomédicas |

**Total: 5 herramientas principales**
